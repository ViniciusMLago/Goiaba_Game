# 🍈 Jogo da Goiabinha

Um jogo 2D no estilo *endless runner* desenvolvido com **Python + Pygame**, inspirado no clássico jogo do dinossauro do Google.

O objetivo é simples: sobreviver o máximo possível, desviar dos obstáculos e coletar bônus para aumentar sua pontuação!

---

## 🎮 Gameplay

- 🏃 Personagem com animação em sprite sheet
- ⬆️ Pulo duplo
- 🚧 Obstáculos variados
- 🎁 Sistema de bônus com efeitos especiais:
  - 🛠️ Ferramenta → Imunidade temporária
  - 👶 Bebê → Pontos extras + mensagem especial
  - 💰 PPR → Grande bônus de pontuação (raro)
- 📈 Pontuação automática crescente
- 🖥️ Tela inicial
- 💀 Tela de Game Over com botão de reinício

## 🚀 Como Executar

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/ViniciusMLago/Goiaba_Game.git
cd Goiaba_Game
```

### 2️⃣ Crie um ambiente virtual (recomendado)
```bash
py -3.11 -m venv venv
venv\Scripts\activate
```

### 3️⃣ Instale as dependências
```bash
pip install -r requirements.txt
```

### 4️⃣ Execute o jogo
```bash
python main.py
```
